Vulnerability: Unauthorized access to physical facilities
Discipline: Physical Security
Sector: Critical Infrastructure

Description: This vulnerability involves the potential for unauthorized individuals to gain access to critical infrastructure facilities through inadequate physical security measures.

Options for Consideration:
1. Implement multi-factor authentication for physical access
2. Install biometric access controls
3. Deploy security cameras with motion detection
4. Establish visitor management protocols
5. Conduct regular security assessments

Source: CISA Physical Security Guidelines, 2023
