# Archive Directory

This directory contains old, unused, or deprecated files that are kept for reference but no longer used in the project.

## Contents

### Old Admin Pages
- `page-old.jsx` - Old version of admin page
- `page-clean.jsx` - Clean version of admin page

### Beast Variants
- `beast-variants/` - Contains all `-Beast.js` variant files that were experimental or alternative implementations

## Note

Files in this directory are **not** imported or used anywhere in the codebase. They are kept for historical reference only.

If you need any of these files, they can be restored by moving them back to their original locations (and updating imports as needed).
